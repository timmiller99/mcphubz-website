Getting Started with MCPHubz API

Follow these steps to start using the MCPHubz API:

1

Sign up for an API Key

Create your free account and get your API key to start making requests.

2

Make Your First Request

Use your API key to fetch the list of available MCP servers.

curl -H "Authorization: Bearer YOUR_API_KEY" \
     -H "Content-Type: application/json" \
     https://api.mcphubz.com/v1/servers
3

Start Building

Integrate the API into your application using our SDKs or direct HTTP requests.

Authentication

All API requests must include authentication using your API key.

API Key Authentication

Include your API key in the Authorization header:

Authorization: Bearer YOUR_API_KEY
Rate Limits
  • Free: 100 requests/day
  • Pro: 10,000 requests/day
  • Enterprise: Unlimited
Error Codes
  • 401: Invalid API key
  • 429: Rate limit exceeded
  • 403: Insufficient permissions

API Endpoints

Comprehensive list of available API endpoints:

Servers

GET /v1/servers

Get a list of all MCP servers

Query Parameters:
  • category - Filter by category (optional)
  • sdk - Filter by SDK (optional)
  • limit - Number of results (default: 10)
  • page - Page number (default: 1)
GET /v1/servers/search

Search for MCP servers

Query Parameters:
  • q - Search query (required)
  • category - Filter by category (optional)
  • limit - Number of results (default: 10)
GET /v1/servers/{id}

Get detailed information about a specific server

Analytics

GET /v1/analytics/popular

Get popular MCP servers with download statistics

Code Examples

Ready-to-use code examples in multiple programming languages:

Fetching MCP Servers

// Fetch MCP servers using JavaScript
const response = await fetch('https://api.mcphubz.com/v1/servers', {
    headers: {
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    }
});

const data = await response.json();
console.log('MCP Servers:', data.servers);
# Fetch MCP servers using Python
import requests

headers = {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
}

response = requests.get('https://api.mcphubz.com/v1/servers', headers=headers)
data = response.json()

print(f"Found {len(data['servers'])} MCP servers")
# Fetch MCP servers using cURL
curl -H "Authorization: Bearer YOUR_API_KEY" \
     -H "Content-Type: application/json" \
     https://api.mcphubz.com/v1/servers

Searching for Servers

// Search for GitHub-related MCP servers
const searchQuery = 'github';
const response = await fetch(`https://api.mcphubz.com/v1/servers/search?q=${searchQuery}`, {
    headers: {
        'Authorization': 'Bearer YOUR_API_KEY'
    }
});

const results = await response.json();
results.servers.forEach(server => {
    console.log(`${server.name}: ${server.description}`);
});
# Search for GitHub-related MCP servers
import requests

search_query = 'github'
url = f'https://api.mcphubz.com/v1/servers/search?q={search_query}'

response = requests.get(url, headers={'Authorization': 'Bearer YOUR_API_KEY'})
results = response.json()

for server in results['servers']:
    print(f"{server['name']}: {server['description']}")

API Endpoints

List Servers

GET /v1/servers

Returns a list of MCP servers with basic information.

Query Parameters
  • page - Page number (default: 1)
  • limit - Results per page (default: 20, max: 100)
  • category - Filter by category
  • sdk - Filter by SDK
  • search - Search term

Get Server Details

GET /v1/servers/{id}

Returns detailed information about a specific MCP server.

Search Servers

GET /v1/search

Search for MCP servers using advanced filtering.

Query Parameters
  • q - Search query
  • filters - JSON object with filter criteria
  • sort - Sort field and direction

Server Analytics

GET /v1/servers/{id}/analytics

Returns usage and popularity metrics for a specific server.

Professional & Enterprise

Bulk Server Details

POST /v1/servers/bulk

Get details for multiple servers in a single request.
